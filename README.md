# 数据抓取分析助手 (Chrome插件板)
### 根据选择元素，自动生成简HTML选择命令，用于爬虫网页内容分析
## 功能：
* 根据选择的元素，自动计算相应的 Jquery selector， 并可以直接用于 python BeautifulSoup等package
* 工具选择的元素，提取文本内容
